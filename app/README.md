# WORD LAB 단어시험지 생성기

브라우저에서 실행하는 데스크톱용 정적 웹앱이며, 별도로 윈도우 실행파일 버전도 함께 만들었습니다.

## 실행파일 버전

- 실행파일 경로: `C:\Users\CHOI\Documents\Playground\Projects\VocabTestGenerator.Desktop\bin\Release\net8.0-windows\win-x64\publish-single\VocabTestGenerator.Desktop.exe`
- 실행파일 버전은 저장 폴더를 앱이 직접 기억합니다.
- 한 번 저장 루트를 고르면 다음 실행 때도 그대로 유지되고, 사용자가 바꿀 때만 변경됩니다.

## 사용 방법

1. `index.html`을 Edge 또는 Chrome에서 엽니다.
2. 바탕화면에 `WORD LAB OUTPUT` 같은 일반 폴더를 하나 만든 뒤, `저장 폴더 연결` 버튼으로 그 폴더를 선택합니다.
3. 반을 추가하고, 각 반마다 `단어[TAB]뜻` 형태로 엑셀 내용을 붙여넣습니다.
4. 출제 방향, 문항 수, 생성 차수를 고릅니다.
5. `PDF 생성 시작`을 누르면 `선택한폴더/YYYY-MM-DD/반이름/반이름_1차.pdf` 형식으로 저장됩니다.
6. 같은 날짜 폴더 안에 `WORD_LAB_단어복원.json`도 함께 생성됩니다.

- 입력한 반 정보와 단어 목록은 다시 열어도 복원됩니다.
- 저장 폴더도 함께 기억하며, 브라우저 권한이 유지된 경우 자동 복원되고 권한이 끊긴 경우 버튼 한 번으로 다시 연결할 수 있습니다.
- `복원 파일 불러오기` 버튼으로 예전에 저장된 `WORD_LAB_단어복원.json`을 다시 읽어 반별 단어와 설정을 복구할 수 있습니다.

## 실행파일 버전 사용 방법

1. `VocabTestGenerator.Desktop.exe`를 실행합니다.
2. `저장 폴더 연결` 버튼으로 저장 루트를 한 번만 선택합니다.
3. 이후에는 앱을 껐다 켜도 같은 저장 폴더가 유지됩니다.
4. PDF와 복원 JSON은 기존과 같은 구조로 자동 저장됩니다.

## 입력 형식

- 가장 권장: `단어[TAB]뜻`
- 추가 지원: `단어 | 뜻`, `단어, 뜻`, `단어  뜻`(두 칸 이상 띄어쓰기)

## 중복 규칙

- 같은 단어가 여러 번 들어오면 뜻을 합쳐 하나로 정리합니다.
- 단어 수가 `문항 수 x 차수`보다 적으면 차수 간 완전 무중복은 불가능합니다.
- 이 경우에도 각 차시는 가능한 한 다르게, 중복이 최소화되도록 랜덤 구성합니다.

## 저장 폴더 주의

- Chromium 계열 브라우저는 `Desktop`, `Documents`, `Downloads`의 루트 자체를 민감 폴더로 막을 수 있습니다.
- 따라서 `Desktop` 자체가 아니라 `Desktop\\WORD LAB OUTPUT` 같은 하위 폴더를 선택해야 정상 동작합니다.
